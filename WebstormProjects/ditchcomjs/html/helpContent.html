• connect: connect to a bluetooth device (chrome only)
  • disconnect: disconnect from the bluetooth device
  • help: display this help menu
  • clear: clear the screen
  • led1 [R] [G] [B] [W]: control LED1 with specified RGBW values (e.g., led1 255 0 0 255)
  • led2 [R] [G] [B] [W]: control LED2 with specified RGBW values (e.g., led2 255 0 0 255)
  • heater [pwm1/pwm2] [dutyCycle]: control heater's PWM duty cycle (e.g., heater pwm1 50)
  • dcdcenable [0/1]: enable or disable the DCDC converter (e.g., dcdcenable 1)
  • pressurevalues: read and display pressure sensor values
  • accelvalues: read and display accelerometer values
  • analogvalues: read and display analog input values
  • digitalvalues: read and display digital input values
  • rtcvalue: read and display the real-time clock value
  • readnfc: read and display NFC tag information
  • save -log <file_name>: save the logs from the information region into a .txt file
  • save -term <file_name>: save the content from the terminal region into a .txt file
  • save -bleinfo <file_name>: save the available connected ble information into a .txt file
  • show -devicename: show device name and address
  • arrow up/down: navigate through the command history
